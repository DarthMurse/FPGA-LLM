Required Data Files
===================

This directory requires model weight binary data files for the TeLLMe accelerator.

Please download the required files from:
https://drive.google.com/drive/folders/1jrOIA7jaPUJkjDhWwoLr_jXlYaViF0vz?usp=sharing

Instructions:
1. Visit the Google Drive link above
2. Navigate to the "weight_bin_5/" folder
3. Download all weight binary files to this directory
4. Ensure file permissions are set correctly (readable by the user)

Note: These are quantized 1.58-bit ternary weights for BitNet b1.58-large model.

For questions or issues, please refer to the main README.md file.
